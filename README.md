- React DOM

- Firestore
- React Bootstrap
- Styled Components
- Sweet Alert
## Levantar el proyecto
- Instalar Node JS
- Abrir CMD
- Posicionarse en la carpeta raiz del proyecto
- Ejecutar el comando por consola "npm install"
- Luego ejecutar "npm start"
- Abrir el buscador y dirigirse a la url http://localhost:3000/
